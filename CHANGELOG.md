## [0.1.0] - 2026-01-28

### Added
- Initial release
- Support for JavaScript/Node.js errors
- Support for Python tracebacks
- Support for Java exceptions
- Support for Go panics
- Support for Rust panics
- Automatic format detection
- Noise removal (timestamps, addresses, UUIDs, hex values, paths)
- Stack frame deduplication
- Verbose output mode
- Read from stdin or file
